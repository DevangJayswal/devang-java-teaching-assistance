api-gateway (localhost:8765)
http://localhost:[api_gateway_port]/[app_name_from_eureka_server]/[url_to_call_on_app]
api_gateway_port=8765
app_name_from_eureka_server=CURRENCY-EXCHANGE
url_to_call_on_app=currency-exchange/from/USD/to/INR
[calling currency-exchange using api-gateway]
http://localhost:8765/CURRENCY-EXCHANGE/currency-exchange/from/USD/to/INR
Initially we will get an error. Saying not found
what we are doing here that we want api-gateway to talk to naming-server with app_name_from_eureka_server
and execute request to above url
However we will have to enable this feature in application.properties Initially
